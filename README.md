Analyzed a large HR dataset (200,000 records) to uncover workforce insights and trends.

Key Steps

Data Cleaning in Jupyter: Used Python and Pandas to clean and preprocess raw HR data. 

Data Visualization: Conducted univariate and bivariate analysis to explore employee demographics, attrition patterns, and performance indicators.

Data Verification: In SSMS using SQL deeper analysis was done. 

Future Extension: Fetching data from SSMS to Power BI for interactive reporting dashboards.

Tools Used

Python (Pandas, Matplotlib, Seaborn)

Jupyter Notebook

SQL Server Management Studio
